##  프로젝트 목표

이스트소프트 백엔드 과정의 **2차 팀 프로젝트**로
커뮤니티 게시판 웹 애플리케이션의 **도메인 모델링과 데이터베이스 설계**를 목표로 한다.

본 프로젝트는 기능 구현 이전 단계에서
**요구사항 명세서와 엔티티 간 관계 정의**에 중점을 둔다.

---

##  설계 대상 도메인

* 사용자(User)
* 게시판(Board)
* 게시물(Post)
* 댓글(Comment)
* 쪽지(Message)

---

## 사용 기술
-Backend : Java, Spring Framework
-Database : H2
-Build : Gradle
